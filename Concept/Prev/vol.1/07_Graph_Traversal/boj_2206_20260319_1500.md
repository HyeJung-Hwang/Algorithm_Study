# BOJ 2206 - 벽 부수고 이동하기 (2026-03-19)

## 핵심 아이디어: BF vs BFS 상태 확장

### BF로 벽 위치 일일이 선택하는 방식 (비효율)
벽마다 BFS를 새로 돌리는 방식 → 너무 느림.

### flag를 상태에 포함시키는 방식 (정답)
- 일반 BFS: `(r, c)` → 위치만 상태
- 이 문제: `(r, c, flag)` → 위치 + 벽 사용 여부가 상태

BFS가 퍼져나가면서 "부수는 경로", "안 부수는 경로"를 **자동으로 동시에 탐색**.

---

## BFS 큐 = 탐색 예정 노드 관리 + 상태

> **"상태" = "이 노드를 탐색하기 위해 필요한 모든 정보"**

`(r, c)` 만으로는 부족한 이유:
- `(2, 3)`에 도달했을 때 벽을 쓰고 온 건지, 안 쓰고 온 건지에 따라 앞으로 갈 수 있는 경우가 달라짐
- 그래서 `(r, c, flag)` 가 필요

---

## 같은 칸이 flag 다르게 큐에 두 번 들어갈 수 있나?

```
(r, c, flag=0)  ← 벽 안 부수고 도달한 경로
(r, c, flag=1)  ← 벽 부수고 도달한 경로
```

→ **맞음.** 두 개는 다른 상태이기 때문. flag=0은 앞으로 벽을 부술 수 있고, flag=1은 못 부수니까 앞으로의 가능성이 다름.

---

## 이동 분기 (조건문 짜는 순서)

flag 먼저 보지 말고, **다음 칸이 뭔지 먼저** 보는 게 직관적:

```
다음 칸이 빈 칸(0) → flag 상관없이 그냥 이동, flag 유지
다음 칸이 벽(1)    → flag == 0 일 때만 부수고 이동, flag → 1
                     flag == 1 이면 이동 불가
```

### 3단계 구조

```
범위 체크          → 가장 바깥
이동 가능 여부     → 중간 (다음 칸 종류 + flag)
미방문 체크        → 큐에 넣기 직전
```

---

## dist 배열 설계

```python
dist = [[[float('inf')] * 2 for _ in range(M)] for _ in range(N)]
```

- `dist[r][c][0]` = 벽 안 부수고 `(r,c)` 까지의 최단거리
- `dist[r][c][1]` = 벽 부수고 `(r,c)` 까지의 최단거리
- `float('inf')` = 미방문 (visited 배열 따로 필요 없음)

> graph(원본) / dist(거리) 는 반드시 분리해서 관리. graph를 덮어쓰면 벽(1)과 거리(1)가 구분 안 됨.

---

## 실수 모음

| 실수 | 원인 |
|---|---|
| `visited`를 2D로 만듦 | 상태가 `(r,c,flag)` 인데 visited도 3D여야 함. 2D면 flag=1 경로를 막아버림 |
| `else` 블록에서 빈칸도 flag=1로 설정 | 빈 칸 이동은 flag 유지, 벽일 때만 flag=1 |
| `graph`를 거리 저장용으로 덮어씀 | 원본 그래프와 dist 배열 분리 필요 |
| `return graph[u][v][flag]` | graph는 2D → `return dist[u][v][flag]` 로 수정 |

---

## 최종 코드

```python
import sys
input = sys.stdin.readline
from collections import deque

dx = [-1,1,0,0]
dy = [0,0,-1,1]

def bfs():
    queue = deque([(0,0,0)])
    dist[0][0][0] = 1

    while queue:
        u,v,flag = queue.popleft()

        if u == N-1 and v == M-1:
            return dist[u][v][flag]

        for i in range(4):
            nx,ny = u + dx[i], v+dy[i]
            if 0 <= nx < N and 0 <= ny < M:  # 범위 체크
                if graph[nx][ny] == 0:        # 빈 칸
                    if dist[nx][ny][flag] == float('inf'):  # 미방문
                        dist[nx][ny][flag] = dist[u][v][flag] + 1
                        queue.append((nx,ny,flag))
                else:                          # 벽
                    if flag == 0:              # 아직 안 부쉈으면
                        if dist[nx][ny][1] == float('inf'):
                            dist[nx][ny][1] = dist[u][v][0] + 1
                            queue.append((nx,ny,1))

    return -1

if __name__ == "__main__":
    N,M = map(int,input().split())
    graph = [list(map(int,input().strip())) for _ in range(N)]
    dist = [[[float('inf')]*2 for _ in range(M)] for _ in range(N)]
    print(bfs())
```
