# BOJ 2589 - 보물섬 (2026-03-21)

## 접근 방식

"육지 두 곳 사이의 최단거리 중 가장 긴 것" → **모든 육지마다 BFS 따로 돌리기**

```
모든 육지 노드 A에 대해:
    A에서 BFS → dist 배열 생성 (A에서 각 칸까지 최단거리)
    → max(dist) 로 가장 먼 육지까지 거리 구하기
    → 전체 최댓값 갱신
```

---

## 멀티소스 BFS vs 시작점마다 BFS

배열에 뭐가 저장되는지로 구분:

| | dist[r][c] 에 저장되는 값 |
|---|---|
| **멀티소스 BFS** | 가장 가까운 시작점까지의 거리 |
| **시작점마다 BFS** | 출발점 A에서 (r,c)까지의 거리 |

- 멀티소스: "나한테 가장 가까운 시작점이 얼마나 멀어?" → 토마토, 바이러스 문제
- 시작점마다: "이 출발점에서 각 칸까지 얼마나 걸려?" → 이 문제처럼 두 점 사이 거리 필요할 때

## 버그 모음

### 버그 1: graph 파싱 오류 (IndexError 원인)

```python
# 잘못됨 - "WLLWWWL" 을 ["WLLWWWL"] 로 읽음
graph = [list(input().strip().split()) for _ in range(N)]

# 올바름 - 문자 하나씩
graph = [list(input().strip()) for _ in range(N)]
```

`split()` 쓰면 공백 기준으로 나눠서 문자열 통째로 들어감 → `graph[n][m]` 에서 IndexError.

### 버그 2: dist를 BFS마다 초기화 안 함

육지마다 BFS를 따로 돌리는데 dist 초기화 안 하면 이전 결과가 남아있음.

```python
for n in range(N):
    for m in range(M):
        if graph[n][m] == "L":
            dist = [[0]*M for _ in range(N)]  # 매번 초기화
            dist[n][m] = 1                     # 시작점 거리 1
            ans = bfs(n, m)
```

### 버그 3: 거리 계산 로직

```python
# 잘못됨 - 항상 0+1=1 이 됨
dist[nx][ny] = dist[nx][ny] + 1

# 올바름 - 이전 칸의 거리 기반
dist[nx][ny] = dist[x][y] + 1
```

## BFS 조건문 3단계 구조

```python
if 0 <= nx < N and 0 <= ny < M:      # 1) 범위 체크
    if graph[nx][ny] == "L":          # 2) 방문 가능 여부
        if dist[nx][ny] == 0:         # 3) 미방문 체크 (넣기 직전)
            dist[nx][ny] = dist[x][y] + 1

```
